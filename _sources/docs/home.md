# 向阳花花农的花海

***The Sunflorist's Ocean of Flowers***

很久很久以前，在一个很远很远的地方，生活着一个无忧无虑的小男孩。他每天都会躺在大草原上，出神地望着那些淘气的云朵，想象它们是太阳牧放的羊群，或是月亮定制的婚纱。

有一天，小男孩和往常一样对着天空幻想着。这时，整个世界渐渐暗淡下来，乌云翻涌，狂风四起，电闪雷鸣，暴雨倾盆。小男孩期盼着天空能够放晴，可是许多天过去了，风越刮越猛，雨越下越大，骇浪滔天，地动山摇。小男孩在风浪里迷失了方向，他紧紧地抱住一根浮木，漂荡着、飘荡着……

当小男孩从昏迷中苏醒过来，他的眼前是一片荒芜的沙漠。他不知道自己睡了多久，也不知道自己身在何处。小男孩唯一能做的就是向前走，尽管分不清哪里是前，尽管走得十分艰难，但他依然没有放弃，依然相信着最终会到达一块绿洲。

恍恍惚惚中，一个绿色的斑点在视野里跳跃着。小男孩用尽最后的一丝力气，一步一步地朝着那里靠近。一棵仙人掌一步一步地长高长大，像天地之间最后的一根支撑。在仙人掌的旁边，小男孩找到了一个小水洼。它看起来随时都有干涸的可能，却总有清凉的水从细沙之下源源不断地渗上来。就这样，小男孩获救了，他从此留在了这里。

那天，一场沙尘暴席卷了整个沙漠。等风沙停了以后，仙人掌下露出了一堆被掩埋多年的麻袋。打开后发现里面有各种花卉的种子，小男孩于是决定将它们种下。有时种子迟迟不发芽，有时花苗因为缺乏养分而长势不佳，还有的时候，花蕾刚绽放不久就葬身沙尘。然而小男孩没有灰心丧气，仍旧努力地种着花，哪怕只有一朵能活下来。

一天又一天过去了，终于，在那棵仙人掌的周围，漫山遍野都开满了花，五彩斑斓，生生不息。

也许小男孩早已不是小男孩了，但每当他静静地看着这片沙漠中的花海时，他还是会热泪盈眶得像个小孩，心花怒放得像个小孩，天真烂漫得像个小孩。
